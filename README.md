# PyBioAnalyzer
Python codes for analyzing bioanalyzer files

# Requirements
Checked compatible with python3.7.
- matplotlib
- scipy 1.2.1
- pandas

# Usage
```
python PyBioAnalyzer.py --in_dir example/ --assay_type HS_DNA --min_lim 30 --max_lim 2000
```
